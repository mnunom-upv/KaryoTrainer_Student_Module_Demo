

#  KaryoTrainer_Student_Module_Demo 🚀

This repository includes an installable demo version of the **KaryoTrainer Student Module** Android app. 

This version has the following limitation: 

- Loads a single image (always the same one) and displays it to the user. 
- Includes a checkbox to display chromosome labels (the final version does not), allowing the user to check their progress. 
- Allows zooming (using Pinch In/Stretch gestures) to zoom in or out of the image, and a swipe gesture to move around once zoomed in or out. Zoom mode must be disabled to switch to chromosome skeleton selection mode. 
- We recommend the chromosome selection not exceed the chromosome’s boundaries and remain as centered as possible within them. Once the skeleton is selected, a dialog box will appear, allowing the user to view the chromosome segmentation based on the selected skeleton (undo is available if the segmentation result is not satisfactory) and select which pair it belongs to (1–23, X, or Y).


## Steps to Install the Android App

- 🚀 **Download** the KaryoTrainer_Student_Module_Demo.apk
- 📁 **Install**: install by doulbe tap on the KaryoTrainer_Student_Module_Demo.apk
- 📁 **Run**: tap on the **KaryoTrainer_Student_Module_Demo** icon in the app list.
